Git bash to Github
    Untuk Clone
  1. git clone (link yang ingin di clone)
    2. Awal
  2. git config --global user.name ""
  3. git config --global user.email ""
  4. git init
  5. git add * / git add .
  6. git remote add origin "linkfolder.github"
  7. git branch -M master
  8. git commit - m "pertama"
  9. git push -u origin master
    10. Kedua
  10. git add * / git add .
  11. git pull origin master
  12. git commit -m "kedua"
  13. git push -u origin master
  14. Untuk fetch yg terupdate
  15. git fetch; git reset --hard origin/master
